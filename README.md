# CodeAphla_language_translation
# 🌐 LinguaAI — Language Translation Tool
> CodeAlpha AI Internship | Task 1

A professional AI-powered language translation web app built with Python & Streamlit, supporting 30+ languages with real-time text-to-speech output.

## ✨ Features
- 🌍 Translate between 30+ languages instantly
- 🔍 Auto-detect source language
- 🔊 Text-to-Speech for translated output
- 📋 Copy translated text easily
- ⚡ Shows translation time & word/char stats
- 🎨 Beautiful dark UI with glassmorphism design

## 🛠️ Tech Stack
| Layer | Tech |
|-------|------|
| Frontend | Streamlit |
| Translation | Google Translate API (`googletrans`) |
| Text-to-Speech | gTTS |
| Deployment | AWS Elastic Beanstalk |
| Language | Python 3.10+ |

## 🚀 Run Locally

```bash
git clone https://github.com/navyagupta0709/CodeAlpha_LanguageTranslation
cd CodeAlpha_LanguageTranslation
pip install -r requirements.txt
streamlit run app.py
```
